## Data Flow

```
User Input (Modal)
    ↓
Form Validation
    ↓
Generate Course ID
    ↓
Create Course Object
    ↓
Save to localStorage
    ↓
Navigate to Builder
    ↓
Builder Loads Data
    ↓
User Builds Course
    ↓
Submit for Review
    ↓
Update Status
    ↓
Back to Dashboard
```

## Key Interactions

### Open Modal
```
Click "Create New Course" 
  → setShowCreateCourseModal(true)
  → Modal renders
```

### Close Modal
```
Click backdrop OR Click X OR Press ESC
  → setShowCreateCourseModal(false)
  → Modal unmounts
```

### Submit Form
```
Click "Create Course"
  → validate()
  → If valid:
    → setIsCreating(true)
    → Generate courseId
    → Create course object
    → Save to localStorage
    → setIsCreating(false)
    → onClose()
    → navigate(`/course-builder/${courseId}`)
  → If invalid:
    → setErrors(validationErrors)
    → Show error messages
```

## Responsive Behavior

### Desktop (> 1024px)
- Modal: 672px width
- Form: 2-column grid for Category/Level
- All fields visible
- Comfortable spacing

### Tablet (768px - 1024px)
- Modal: 90% width
- Form: 2-column grid maintained
- Slightly reduced spacing

### Mobile (< 768px)
- Modal: 95% width with margins
- Form: Single column (Category/Level stack)
- Reduced padding
- Larger touch targets
