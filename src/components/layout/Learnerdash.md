Design a modern, responsive learner dashboard for an online learning platform focused on microlearning, progress tracking, and achievement.

Visual Style

Use a deep blue monochromatic theme with the base color #1E2348, complemented by orange accents (#FF6B4D) for primary actions and highlights.
The design should feel:

professional

clean and structured

motivating and progress-driven

Use Poppins as the primary font and maintain strong visual hierarchy.

Layout Requirements
Top Navigation

Include:

Platform logo

Search bar

Notifications icon

User profile avatar dropdown

Left Sidebar Navigation

Include these sections:

Dashboard

My Courses

Microlearning Paths

Assessments

Assignments & Capstones

Certificates & Badges

Discussions

Help Center

Sidebar should use indigo tones with orange highlight for the active item.

Main Dashboard Content
1. Welcome Section

Display:

Greeting with learner’s name

Quick summary of learning progress

“Resume Learning” primary orange CTA button

2. Current Course Progress

Show:

Current course card

Progress bar

Resume button

Last lesson accessed

3. Microlearning & Daily Challenges

Display:

Daily challenge card

Streak indicator

Microlearning path suggestions

4. Achievements & Gamification

Include:

Badges earned

Certificate progress

Leaderboard preview

5. Upcoming Activities

Include:

Upcoming live classes

Assignment deadlines

Assessment reminders

6. Learning Insights

Show:

Weekly learning time

Lessons completed

Quiz performance summary

Use charts or visual progress indicators.

Functional Modules to Represent

Ensure the design visually accommodates:

Learning player and course outline

Audio lesson support

In-lesson notes and learning notes

Quizzes and assessment feedback

Discussion forums and class chat

These do not all need to be on one screen but should be represented in the dashboard ecosystem.

UX Expectations

Clear visual hierarchy

Card-based layout

Strong use of whitespace

Orange used only for actions and highlights

Fully responsive (desktop-first, adaptable to tablet)

Output

Generate:

A high-fidelity dashboard UI

Light theme

Clean, SaaS-style layout suitable for a digital learning platform
