Once configured, test each provider:

4️⃣ What to Look For

  • Success: Redirects to provider login page
  • Success: After login, redirects back to /learning
  • Success: Profile is auto-created in database
  • Success: User info appears in navbar

5️⃣ Common Issues

  • "Invalid redirect URI" → Check callback URL matches exactly
  • Button does nothing → Provider not enabled in Supabase
  • "Failed to sign in" → Check Client ID/Secret are correct
  • Profile not created → Check browser console for errors
